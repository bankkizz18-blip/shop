# 🎬 NetflixRent - คู่มือการใช้งาน

## 📁 โครงสร้างไฟล์

```
webshopnetflix/
├── inedex.html          # หน้าเว็บหลัก (ลูกค้าใช้)
├── wheel.html           # หน้าวงล้อเสี่ยงโชค
├── admin.html           # หน้าจัดการเว็บไซต์ (Admin)
├── script.js            # ระบบ JavaScript ทั้งหมด
├── style.css            # สไตล์/ความสวยงาม
├── Code.gs              # Google Apps Script (Backend - ถ้าใช้)
└── README.md            # คู่มือนี้
```

## 🚀 เริ่มต้นใช้งาน (3 ขั้นตอน)

### 1. เปิดหน้าเว็บหลัก
```
เปิดไฟล์: inedex.html
```
- นี่คือหน้าที่ลูกค้าจะเห็น
- มีแพ็กเกจ, รีวิว, วงล้อ, ติดต่อ

### 2. เปิดหน้า Admin (จัดการ)
```
เปิดไฟล์: admin.html
```
- จัดการรีวิว (ลบ, ดูสถิติ)
- ดูแดชบอร์ด
- สำรองข้อมูล

### 3. ทดสอบ
1. เปิด `inedex.html` → เขียนรีวิว
2. เปิด `admin.html` → จะเห็นรีวิวโผล่มาทันที!
3. ใน Admin → ลบรีวิวได้
4. กลับไปหน้าหลัก → รีวิวหายไป!

## 🎯 การใช้งานประจำวัน

### เพิ่มรีวิว (หน้าเว็บหลัก)
1. ลูกค้ากดปุ่ม "เขียนรีวิว"
2. กรอกชื่อ, รีวิว, ให้คะแนน
3. กด "ส่งรีวิว"
4. รีวิวจะแสดงทันที

### ลบรีวิว (หน้า Admin)
1. เปิด admin.html
2. ไปที่แท็บ "จัดการรีวิว"
3. กดปุ่ม "ลบ" ที่รีวิวที่ต้องการ
4. หรือกด "ลบรีวิวทั้งหมด" เพื่อล้างทั้งหมด

### ดูสถิติ (หน้า Admin)
1. เปิด admin.html
2. แท็บ "แดชบอร์ด" จะแสดง:
   - จำนวนรีวิวทั้งหมด
   - คะแนนเฉลี่ย
   - จำนวนลูกค้าไว้วางใจ
   - จำนวนแพ็กเกจ

### ใช้ Google Apps Script (ข้อมูลบน Cloud)

**ขั้นตอน:**
1. ไปที่ https://script.google.com/
2. สร้างโปรเจคใหม่
3. คัดลอกโค้ดจาก `Code.gs` ไปวาง
4. Deploy > Web App > Who has access: Anyone
5. คัดลอก Web App URL
6. เปิด admin.html > ตั้งค่า > วาง URL > บันทึก
7. แก้ไข `script.js` บรรทัด 147:
   ```javascript
   const USE_APPS_SCRIPT = true;
   ```

**ประโยชน์:**
- ข้อมูลแชร์ระหว่างผู้ใช้ทุกคน
- ข้อมูลไม่หายเมื่อเคลียร์ cache
- ดูข้อมูลใน Google Sheet ได้

**ข้อจำกัด:**
- ต้องตั้งค่าเพิ่ม
- มี quota (5,000 เขียน/วัน, 50,000 อ่าน/วัน)

## 💾 การสำรองข้อมูล

### Export (สำรองข้อมูล)
1. เปิด admin.html
2. แท็บ "ตั้งค่า"
3. กด "สำรองข้อมูล (Export)"
4. ไฟล์ JSON จะดาวน์โหลดมา

### Import (กู้ข้อมูล)
1. เปิด admin.html
2. แท็บ "ตั้งค่า"
3. กด "กู้ข้อมูล (Import)"
4. เลือกไฟล์ JSON ที่สำรองไว้
5. ข้อมูลจะถูกรีโหวต

### รีเซ็ตข้อมูลทั้งหมด
1. เปิด admin.html
2. แท็บ "ตั้งค่า"
3. กด "รีเซ็ตข้อมูลทั้งหมด"
4. ยืนยัน
5. ข้อมูลจะกลับสู่ค่าเริ่มต้น

## 🔧 แก้ปัญหา

### รีวิวไม่แสดง
1. กด F12 เปิด Console
2. ดูว่ามี error อะไร
3. ลอง refresh หน้าเว็บ (Ctrl+Shift+R)

### ข้อมูลไม่ sync ระหว่างหน้า
1. ตรวจสอบว่าเปิดทั้ง 2 หน้าจากโฟลเดอร์เดียวกัน
2. ลอง refresh ทั้ง 2 หน้า
3. ตรวจสอบ Console มี error หรือไม่

### Admin เปิดไม่ได้
1. ตรวจสอบว่าเปิดไฟล์ admin.html จากโฟลเดอร์ webshopnetflix
2. ตรวจสอบว่ามีไฟล์ script.js ในโฟลเดอร์เดียวกัน
3. ลองใช้ browser อื่น

### วงล้อไม่ทำงาน
1. ตรวจสอบว่ามีไฟล์ wheel.html
2. เปิด Console ดู error
3. ลอง refresh (Ctrl+Shift+R)
